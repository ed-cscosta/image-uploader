## :books: Instalação

### Instalação - Backend

Basta fazer clone do projecto, instalar as dependências e configurar as **variáveis de ambiente**.

```bash
cd backend
npm install
//config .env variables
npm start
```

### Instalação - Frontend (WEB)

Para iniciar o **Frontend** do React utilize os comandos:

```bash
cd aulas
cd frontend
npm install
//config o endpoint
npm serve
```
